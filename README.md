# Khóa Học Giá Hời - Website Bán Khóa Học Online

## Giới thiệu

Website bán khóa học trực tuyến với giao diện hiện đại, thân thiện người dùng. Trang web cung cấp các khóa học đa dạng từ lập trình, AI, ngoại ngữ với mức giá ưu đãi.

## Tính năng

- **Banner Slideshow**: Hiệu ứng trượt mượt mà với auto-play và điều khiển thủ công
- **Danh mục khóa học**: Phân loại theo chủ đề (Lập trình, AI, Thiết kế, Ngoại ngữ...)
- **Card sản phẩm**: Hiển thị thông tin khóa học với hình ảnh, giá, số học viên
- **Header cố định**: Thanh tìm kiếm và menu điều hướng luôn hiển thị

## Công nghệ sử dụng

- HTML5
- CSS3 (Flexbox, Grid, Transitions)
- JavaScript (Vanilla JS)
- Font Awesome Icons

## Cách chạy

1. Clone repository về máy
2. Mở file `index.html` bằng trình duyệt web
3. Hoặc sử dụng Live Server trong VS Code

## Ảnh minh họa

Website bao gồm:

- Header với logo, thanh tìm kiếm, và các nút hành động
- Menu điều hướng theo danh mục
- Banner slideshow với 3 slides
- Các section khóa học theo chủ đề
- Footer với thông tin liên hệ và mạng xã hội
